# Reroll Dark Magic VFX

Derived from the `Smoke` and `Dizzy` frame sequences in **Free Stylized Sprite VFX** by Kalponic Studio.

- Source: https://kalponic-studio.itch.io/free-stylized-sprite-vfx
- License: Creative Commons Attribution 4.0 International
- License URL: https://creativecommons.org/licenses/by/4.0/

Changes made for this project:

- Recolored the smoke frames to a dark violet/black palette.
- Recolored the dizzy ring frames to a muted arcane violet palette.
- Built an additional dark veil frame sequence from the modified smoke frames to cover dice during rerolls.
- Used the modified frame sequences as a dice reroll overlay animation.
