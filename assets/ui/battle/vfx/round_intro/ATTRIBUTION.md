# Round Intro VFX

Uses selected PNG particles from **Particle Pack (1.1)** by Kenney Vleugels.

- Source: https://www.kenney.nl/assets/particle-pack
- Godot Asset Library: https://godotengine.org/asset-library/asset/784
- License: Creative Commons Zero, CC0
- License URL: http://creativecommons.org/publicdomain/zero/1.0/

Files used in this project:

- `flare_01.png`
- `light_01.png`
- `magic_02.png`
- `spark_05.png`
- `star_02.png`
- `trace_04.png`

The textures are animated and tinted at runtime as the round intro banner VFX.
